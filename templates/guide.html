macOS · Intel & Apple Silicon

Install AIM on a Mac

Since Apple doesn't make old AIM for modern Macs, we'll use a free community project that wraps the real, original AIM software so it runs natively on your Mac.

  1. Go to the im-for-macos releases page and download the newest .zip file:
    github.com/mk6i/im-for-macos/releases
  2. Find the downloaded file in your Downloads folder and double‑click it to unzip. You'll get a file called AOL Instant Messenger.app.
  3. Since this app isn't from the Mac App Store, macOS will block it until you clear one flag. Open the Terminal app and paste this, then press Return: sudo xattr -d com.apple.quarantine ~/Downloads/AOL\ Instant\ Messenger.app It will ask for your Mac password (nothing will appear as you type — that's normal). This step only unblocks this one app; it's safe.
  4. Drag AOL Instant Messenger.app into your Applications folder.
  5. Open it from Applications. The first time, it will ask permission to download the real AIM installer — click Yes and wait a minute for it to finish.
  6. The AIM installer wizard will pop up. Click through it, keeping all the default options.
  7. AIM will open to the sign‑on screen. Click Setup, then choose Connection under Sign On/Off. Enter {{ oscar_host }} as the Host and {{ oscar_port }} as the Port, then click OK.
  8. Type in your screen name and password, and click Sign On — you're in!

Windows 10 / 11

Install AIM on Windows

We'll install a genuine old copy of AIM (version 5.9) and tell Windows to run it in a compatibility mode so it behaves like it did in the 2000s.

  1. Download the compatible client: AIM 5.9.6089.
  2. Run the installer and finish setup, then close AIM completely.
  3. Open the Task Manager (right‑click the taskbar → Task Manager) and make sure AOL Instant Messenger (32 bit) isn't still running. If it is, click it and choose End Task.
  4. Open File Explorer and go to: C:\Program Files (x86)\AIM Find the file aimapi.dll and delete it.
  5. Right‑click aim.exe in that same folder → Properties → Compatibility tab. Check Run this program in compatibility mode for and choose Windows XP (Service Pack 3). Click OK.
  6. Open AIM. At the sign‑on screen, click Setup, then Connection under Sign On/Off. Enter {{ oscar_host }} as the Host and {{ oscar_port }} as the Port, then click OK.
  7. Enter your screen name and password, and sign on.

Web browser

Is there a web version?

No — and it's not really possible to add one. AIM doesn't speak the language web pages use (HTTP); it uses an older messaging protocol from the 1990s called OSCAR, built for a standalone app, not a browser tab. There's no page to open — you'll need to install one of the apps on this page.

iOS & Android

What about my phone?

No confirmed option

We looked for a phone app that can connect to a private AIM server like ours. Nothing currently on the App Store or Play Store is confirmed to work.

The closest lead is Trillian, a chat app with iOS, Android, and web versions that has historically supported AIM. We couldn't confirm whether its current version lets you point it at a custom server instead of Trillian's own network — it's a real long shot, not a recommendation. Old apps like BeejiveIM technically once supported AIM too, but appear abandoned and unmaintained. For now, the desktop apps above are the reliable path.

Not affiliated with AOL — this runs on open-source software (open-oscar-server) that speaks the same protocol the original AIM did.
